Purpose of the program:
The purpose of the program is to check a set of documents for similarities and return a list in order of the pairs of files that have over 200 matching 6-word sequences.

How to use it:
1. Open linux command window
2. Type ‘module load gcc’
3. Unzip the folder and navigate to it
4. Type ‘make’
5. Run the resulting file

What works:
Successfully sorts pairs of plagiarized documents in order for small, medium, and big sets

What doesn’t work:
Nothing

Bugs:
None
